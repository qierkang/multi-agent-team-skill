## 创建与复用规则

- 主线程是唯一允许创建、复用、冻结和归档长期线程的控制面。
- 相同 `domain_key` 不得存在两个活跃长期线程。
- 长期线程内部可以创建一次性子智能体，但不得继续创建长期线程。
- 外部发布、生产写入和凭据变更始终需要用户明确批准。
